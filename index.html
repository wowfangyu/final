第7題(百萬抽獎)

複習ECommerce重點

1.美國2025年元旦發生多起汽車爆炸事件，歹徒到Turo租車，請問Turo屬於哪些類型的電商經營模式？
P2P、O2O
2.我到華碩官方的旗艦店網站買筆電，請問這屬於什麼類型的電商經營模式？
B2C
3.我到蝦皮購買充電寶，到蝦皮店到店取貨，請問這屬於什麼類型的電商經營模式？
O2O
4.天貓，淘寶，分別屬於什麼類型的電商經營模式？
天貓屬於B2C，淘寶屬於C2C
5.footpanda外賣平台屬於什麼類型的電商經營模式？
O2O
6.你的電商要自建網站，網站要雲端委管，請問雲端託管的英文？
Cloud Hosting
請問，目前全世界最大的前三大雲端託管品牌是哪些平台？
AWS.Azure.GCP
7.本學期你做的網站hw1~hw5，請問你上傳到什麼平台來做雲端託管？
github
8.由搜尋引擎（google，yahoo，bing...）而來到你的電商的流量，專業術語稱為什麼流量？（請寫中文，及英文）
自然流量（Organic Traffic）
9.電商最後要處理【物流，金流】，請寫【物流，金流】的英文
物流 ：Logistics
金流 ：Payment Flow
10.請寫出上課提到過的，增加SEO效果的6種實務做法？
1. 進行關鍵字研究
2. 自然融入關鍵字
3. 強調內容的實用性與價值
4. 優化圖片
5. 技術性 SEO
6. 建立外部連結
